Request / Data Flow

  1. User adjusts a filterFilterBar.vue updates shared refs in useFilters (time period · warehouse · category · order status).
  2. View reacts & calls API clientA watch in the active view triggers api.js, which serialises non-all filters to query params.
  3. axios → FastAPI over CORSGET http://localhost:8001/api/...?warehouse=...&category=... — no Vite proxy; direct cross-origin call.
  4. FastAPI filters in-memory dataEndpoint reads module-level lists from mock_data.py, applies query-param predicates, no DB I/O.
  5. Pydantic validates & serialisesResponse models (InventoryItem, Order, DemandForecast, …) shape the JSON payload.
  6. Vue stores raw data in refse.g. allOrders, inventoryItems; computed() derives charts, totals, and table rows reactively.
  7. Template re-rendersSVG charts, CSS-Grid tables, and KPI cards update automatically from the computed graph.
